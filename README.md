# 🕹️ Pong Game (Built with SFML & C++)

A beginner-friendly recreation of the classic **Pong** game using **SFML (Simple and Fast Multimedia Library)** and **C++**. This project is designed to be the perfect first step into game development — combining simplicity with foundational learning.

---

### 🎯 Game Elements
- 🟦 Two Paddles (Player-controlled)
- ⚪ One Ball (With collision & physics)
- 🧮 Score Display (Basic UI logic)

---

## 📚 What I Learn

By building Pong, you'll cover essential topics such as:

- ✅ Drawing shapes and sprites using SFML
- ✅ Handling real-time player input
- ✅ Implementing collision detection
- ✅ Managing game logic and state
- ✅ Updating and displaying score

These principles form the **foundation for more complex games** and even data structure visualizations in SFML.

---

## 🧱 Project Goals

This project is not just about making a game — it's about:

- Building a **strong SFML foundation**
- Learning **structured game loop design**
- Applying **C++ logic in interactive systems**

---

## 🛠️ Technologies Used

- **Language:** C++
- **Graphics Library:** SFML (Simple and Fast Multimedia Library)
- **IDE/Editor:** Visual Studio
